# Assets Directory

This directory contains app assets like icons and splash screens.

## Required Assets

For a production build, you'll need:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen (2048x2732px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

## Temporary Assets

For development, Expo will use default placeholders if these files don't exist.

## Creating App Icon

You can create a simple icon using online tools:
1. Visit https://www.canva.com or similar
2. Create 1024x1024px design with your farm logo
3. Export as PNG
4. Save as `icon.png` in this directory

## Or Use Default Expo Icons

The app will work with Expo's default icons for prototype/demo purposes.
